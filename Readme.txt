1.0 versija

Programma var :
-Kompilēt regex tekstu par automātu
-Ar automātu pārbaudīt vai tas akceptē kādu teksta ievadi
-Saglabāt automātu kā attēlu

Programmai tuvākajai nākotnei plānots:
-Pārbaudīt regex sintakses paireizību
-Atvieglot kompilēšanu saņemot vienu objektu nevis atsevišķi automātu un akceptējošos stāvokļus
